About the Project

This is one of the projects I take full responsibility when I'm working at Disney.

The Community Watch site is envisioned to be a centralized location for safety and security content that is currently posted across several different internal communication channels in various regions. A consolidated site would allow for a centralized repository for content (text, video, etc.), would allow for a centralized location for branding, would provide new opportunities for engagement and interaction, and allow Castmembers to get involved in a number of ways.
